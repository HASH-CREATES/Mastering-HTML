What is HTML?

HTML (HyperText Markup Language) is the skeleton of every web page. In this chapter we learn tags, attributes and how browsers turn code into visual pages.

We will build headings, paragraphs, lists, images and simple forms—no prior experience required.

Practical Activities

  • Create your first index.html file.
  • Experiment with <h1> – <h6>, <p>, <img>.
  • Build an unordered list of your hobbies.
  • Add links to your favourite educational sites.
  • Receive instant feedback via the form below.

About This Module

Designed for Grade 8–10 Computer Science classes, this module aligns with the national curriculum.

After completing the activities you will be able to:

  • Read and write basic HTML.
  • Validate code using the W3C validator.
  • Understand the separation of structure (HTML) and presentation (CSS).
